# Thomas Myers / Anatomy Trains

## Role In Workout OS

Thomas Myers and Anatomy Trains are useful as the myofascial-continuity and chain-based movement influence. Workout OS should use this lens to avoid treating the body as isolated parts only.

## Source Links

- Anatomy Trains history: https://www.anatomytrains.com/about-us/history/
- Anatomy Trains official site: https://www.anatomytrains.com/

## Core Domains

- Myofascial lines.
- Structural integration.
- Regional interdependence.
- Whole-body movement relationships.
- Stability/mobility relationships across chains.

## Principles To Extract

1. **Pain or restriction may not originate where it appears**
   - Hip restriction can affect back mechanics; ankle restriction can affect knees; thoracic stiffness can affect shoulders.
   - Workout OS translation: movement screens look above and below the complaint site.

2. **Movement happens through chains**
   - Pull-ups, carries, crawling, squats, and footwork are full-body patterns.
   - Workout OS translation: exercise cues include whole-body connections, not only target muscles.

3. **Mobility and strength should integrate**
   - Isolated release is less useful if it does not change movement.
   - Workout OS translation: test -> correct -> retest loops.

## Workout OS Rules

- Mobility screens should consider ankle, hip, thoracic spine, shoulder, and neck relationships.
- Warm-ups should include both local prep and chain integration.
- Crawls, carries, and rotational work are core system movements, not novelty.

## Original Example Sequence

**Posterior-chain integration**

```text
Test:
Hip hinge reach and bodyweight squat.

Correct:
1. Hip flexor stretch with posterior tilt
2. Tibialis/soleus work
3. Glute bridge
4. Cross-body crawl

Retest:
Hinge and squat again.

Rule:
If the global pattern improves, the correction belongs in the warm-up or morning session.
```

## What Not To Copy

- Do not reduce Anatomy Trains to vague fascia language.
- Do not make unsupported claims that a single release "fixes" distant pain.
- Use the chain model as a reasoning tool, not as a diagnosis.
