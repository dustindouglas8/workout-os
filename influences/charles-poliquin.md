# Charles Poliquin

## Role In Workout OS

Charles Poliquin is useful as the tempo, structural balance, and disciplined overload influence. Workout OS already uses tempo notation and can use this lineage to make hypertrophy and strength work more precise without needing maximal loads.

## Source Links

- Poliquin Group: https://www.poliquingroup.com/

## Core Domains

- Tempo prescriptions.
- Structural balance.
- Strength and hypertrophy programming.
- Unilateral work.
- Training density and overload.

## Principles To Extract

1. **Tempo changes the stimulus**
   - Eccentric time, pauses, and controlled range can overload tissue without heavier equipment.
   - Workout OS translation: every main lift can have phase-specific tempo options.

2. **Structural balance matters**
   - Weak links and asymmetries eventually cap performance.
   - Workout OS translation: unilateral work, rotator cuff, rear delt, tibialis, soleus, and adductors stay in the system.

3. **Load must be meaningful**
   - Too-light training does not create the adaptation claimed.
   - Workout OS translation: RIR and progression gates prevent endless easy work.

4. **Exercise order matters**
   - Most neurologically demanding work comes before accessories.
   - Workout OS translation: immediate warm-up -> primary block -> support/accessory block -> optional finisher.

## Workout OS Rules

- Include tempo notation in exercise cards.
- Use unilateral movements to expose side-to-side differences.
- Use slow eccentrics as a home-equipment overload mechanism.
- Progress tempo before adding risky complexity.

## Original Example Sequence

**Tempo overload without new equipment**

```text
Goblet Squat Block:
Week 1: 3x10 @ 3-1-1-0
Week 2: 3x10 @ 4-1-1-0
Week 3: 3x8 @ 4-2-1-0
Test week: reduce volume, evaluate depth and knee tracking

Rule:
Tempo only counts if position stays clean.
```

## What Not To Copy

- Do not copy named programs or old-school prescriptions blindly.
- Do not turn tempo work into sloppy slow reps.
- Do not use structural balance claims as medical diagnosis.
