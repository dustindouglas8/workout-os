# Kelly Starrett / The Ready State

## Role In Workout OS

Kelly Starrett is useful as the position-first mobility and durability influence. Workout OS should use this lens to decide whether a user owns the positions required by the program before adding load, speed, or complexity.

## Source Links

- The Ready State: https://thereadystate.com/
- The TRS Difference: https://thereadystate.com/the-trs-difference/

## Core Domains

- Mobility for performance and pain reduction.
- Position standards.
- Daily maintenance dose.
- Tissue quality, joint range, and lifestyle factors.
- Practical self-assessment and correction.

## Principles To Extract

1. **Position before load**
   - Do not add intensity to positions the user cannot access or control.
   - Workout OS translation: phase gates include movement-quality standards, not just reps.

2. **Daily maintenance beats occasional rescue work**
   - Small daily inputs compound better than long, sporadic mobility sessions.
   - Workout OS translation: morning rehab/prehab is low-effort and repeatable.

3. **Mobility should transfer to real movement**
   - The point is not stretching for its own sake; the point is better squats, hinges, pressing, running, and living.
   - Workout OS translation: each mobility drill maps to a movement pattern.

4. **Pain is information**
   - Pain does not automatically mean stop all movement, but it does mean change position, dose, or pattern.
   - Workout OS translation: pain location and score feed substitution and recovery rules.

5. **Lifestyle is part of movement quality**
   - Sitting, breathing, sleep, walking, hydration, and recovery affect positions.
   - Workout OS translation: daily logs and recovery model include lifestyle inputs.

## Workout OS Rules

- Add position checks before progressing load.
- Map mobility drills to squat, hinge, push, pull, gait, or overhead capacity.
- Include short "test -> correct -> retest" loops in mobility modules.
- Make mobility practical and time-capped.

## Original Example Sequence

**Lower-body position reset**

```text
Test:
Bodyweight squat, note ankle/knee/hip restriction.

Correct:
1. Couch-free hip flexor stretch — 40s/side
2. Calf/ankle rock with knee tracking — 10/side
3. 90/90 hip switch — 6/side

Retest:
Bodyweight squat again.

Rule:
If position improves, train. If sharp pain persists, regress lower-body loading.
```

## What Not To Copy

- Do not copy paid pain protocols or full daily videos.
- Do not use mobility work as a reason to avoid strength training.
- Avoid vague "mobilize everything" prescriptions; each drill needs a transfer target.
