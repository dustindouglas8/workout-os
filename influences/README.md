# Influences Index

## Purpose

This folder captures the coaches, authors, and systems that inform Workout OS. The goal is high-fidelity principle extraction, not copying programs.

Each influence page should answer:

- What is this expert/system unusually good at?
- Which principles should Workout OS borrow?
- What should Workout OS avoid copying or overapplying?
- How do those principles become rules, modules, cues, progressions, or tests?
- What original example sequences illustrate the principle without reproducing paid routines?

## Guardrails

- Do not paste paid program workouts, tables, or proprietary progressions.
- Do not quote long passages from books, programs, or paid portals.
- Prefer principles, decision rules, and original examples.
- If an idea comes from a source, include a source link and mark whether it is official, book-derived, or interpretation.
- Workout OS should remain its own system.
