## Principles To Extract

1. **Train muscles without ignoring joints**
   - Chest, shoulders, and back work must respect scapular motion, humeral position, and pain signals.
   - Workout OS translation: pressing and pulling exercises include scapular cueing, pain flags, and regressions.

2. **Pain changes the exercise, not just the effort**
   - Sharp pain is not solved by "go lighter" if the pattern is wrong.
   - Workout OS translation: pain-modified substitutions are first-class fields in `exercises.md`.

3. **Athletic training is multi-plane**
   - Rotation, anti-rotation, carries, crawling, footwork, and conditioning matter.
   - Workout OS translation: include transverse-plane and locomotion work in skill/core days.

4. **Correctives should transfer to main lifts**
   - Rotator cuff, serratus, lower trap, hip, and core drills should support the main session.
   - Workout OS translation: warm-up cards should explain what the activation transfers to.

5. **Supplements are optional, not the foundation**
   - Program design, nutrition consistency, and execution matter more than product reliance.
   - Workout OS translation: supplements stay optional config entries, not universal requirements.

## Workout OS Rules

- Every push day includes shoulder/scapular preparation.
- Every pull day includes scapular depression/retraction prep.
- Pain-modified substitutions should be available before the user starts training.
- Red/yellow readiness should rewrite the workout, not just display advice.

## Original Example Sequence

**Shoulder-safe push primer**

```text
1. Band Pull-Apart — 12-15 controlled reps
2. Scapular Push-Up — 8-10 reps
3. Wall Slide or Serratus Reach — 8-10 reps
4. First push exercise ramp set — 50% effort, perfect shoulder position

Rule:
If anterior shoulder pain appears during primer, switch pressing to push-up variation or landmine-style angle if equipment exists.
```

## What Not To Copy

- Do not reproduce ATHLEAN-X paid workout calendars or exercise lists.
- Avoid making every session a novelty circuit.
- Use joint-aware principles without overcorrecting every normal soreness signal.
