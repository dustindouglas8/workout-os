# Yuri Verkhoshansky

## Role In Workout OS

Yuri Verkhoshansky is useful as the block periodization, special strength, and explosive-strength lineage influence. Workout OS uses this as the logic behind sequencing blocks and phases so each phase prepares the body for the next.

- Special Strength Training reference page: https://complementarytraining.com/special-strength-training-manual-for-coaches/

- Block periodization.

- Long-delayed training effect.
- Plyometric and explosive-strength theory.

## Principles To Extract

1. **Phases should build substrates**
   - A later phase should depend on adaptations created in an earlier phase.
   - Workout OS translation: tissue capacity -> density/work capacity -> strength/specialization.

2. **Special strength bridges general training and performance**
   - Training should become more specific as the athlete earns it.
   - Workout OS translation: general squat/hinge/push/pull becomes skill-specific power, carries, crawls, or ring work later.

3. **Explosive work requires preparation**
   - Tendons, joints, and landing mechanics must be ready before harder plyometrics.
   - Workout OS translation: pogo and ankle stiffness gates precede higher-impact footwork.

4. **Delayed adaptation matters**
   - A phase may pay off after volume is reduced.
   - Workout OS translation: deload/test weeks are part of adaptation, not interruptions.

## Workout OS Rules

- Annual roadmap uses blocks with clear adaptation targets.
- Phase gates decide whether the next training emphasis is appropriate.
- Explosive work is introduced after movement-quality and tissue-capacity gates.
- Do not stack high-intensity blocks without recovery/test checkpoints.

## Original Example Sequence

**Foundation-to-power lower-body pathway**

```text
Phase 1:
Goblet squat tempo + tibialis/soleus + low pogo volume

Phase 2:
Goblet squat density + split squat + controlled landing mechanics

Phase 3:
Jump squat complex or footwork power only if ankle/knee gates pass
```

## What Not To Copy

- Do not paste Soviet templates into a home-training program.
- Do not add shock-method plyometrics before the user has tendon capacity.
- Do not use advanced terminology without turning it into simple rules.
