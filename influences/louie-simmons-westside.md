# Louie Simmons / Westside Barbell

## Role In Workout OS

Louie Simmons and Westside Barbell are useful as the conjugate/concurrent strength influence: rotate stressors, train multiple qualities, identify weak links, and use accessories to build the main patterns.

## Source Links

- Westside Conjugate Method: https://www.westside-barbell.com/pages/conjugate-method
- Westside About: https://www.westside-barbell.com/pages/about

## Core Domains

- Conjugate strength training.
- Max effort, dynamic effort, and repeated effort concepts.
- Weak-point training.
- Exercise variation.
- GPP and accessory volume.

## Principles To Extract

1. **Train multiple qualities without maxing everything**
   - Strength, speed, hypertrophy, and work capacity can coexist when stress is organized.
   - Workout OS translation: heavy/support split, Zone 2 recovery, skill days, and accessories all have distinct roles.

2. **Variation prevents stagnation and overuse**
   - Variation should preserve the movement pattern while changing stress.
   - Workout OS translation: rotate tempo, range, loading, and implements instead of random exercise novelty.

3. **Weak points drive accessories**
   - Accessories exist to improve the system's limiting factor.
   - Workout OS translation: exercise library includes "gap fixed" and "drop first if Yellow."

4. **Repeated effort builds the base**
   - Hypertrophy and local capacity support heavy work.
   - Workout OS translation: support work is lower risk, higher control, and hypertrophy-friendly.

## Workout OS Rules

- Upper Alternating Emphasis uses one heavy pattern and one support pattern.
- Accessories are selected from tested weak points, not preference alone.
- Rotate stressors every phase or block, not every workout.
- Home conjugate means tempo, unilateral work, rings, vest, KBs, and sliders, not imitation powerlifting.

## Original Example Sequence

**Home-conjugate upper day**

```text
Primary Effort:
Ring Push-Up heavy variation — 3-4 hard sets

Repeated Effort Support:
Ring Row — 2-3 controlled sets
Band Pull-Apart — 2 light sets
DB Lateral Raise — 2 hypertrophy sets

Rule:
Heavy pattern drives the day. Support work fixes weak links and should not bury recovery.
```

## What Not To Copy

- Do not copy Westside powerlifting templates into a bodyweight/home context.
- Do not use max-effort strain before technical quality is stable.
- Do not rotate so often that progress cannot be measured.
