# Functional Range Conditioning / Functional Anatomy Seminars

## Role In Workout OS

Functional Range Conditioning is useful as the active mobility and joint-control influence. Workout OS should use FRC-style principles to maintain usable range and make mobility measurable.

## Source Links

- FRC principles: https://functionalanatomyseminars.com/frs-system/functional-range-conditioning/fr-conditioning-principles/
- Functional Anatomy Seminars: https://functionalanatomyseminars.com/

## Core Domains

- Active usable range of motion.
- Joint control.
- CARs: controlled articular rotations.
- Mobility plus strength/resilience.
- Neurological ownership of range.

## Principles To Extract

1. **Range must be active to be useful**
   - Passive flexibility is incomplete if the user cannot control the range.
   - Workout OS translation: mobility benchmarks include controlled movement, not just static positions.

2. **Daily joint mapping matters**
   - Controlled rotations can maintain awareness and reveal daily restrictions.
   - Workout OS translation: morning rehab/prehab can include low-dose CARs for hips, shoulders, neck, and ankles.

3. **End-range control should be earned gradually**
   - Do not force end ranges under load before control exists.
   - Workout OS translation: use phase gates for deep squat, overhead, bridge, and skin-the-cat progressions.

## Workout OS Rules

- Add CARs as optional morning/session prep for joints relevant to the day.
- Track active range quality during phase tests.
- Use controlled range before loaded range.
- Painful pinching changes the drill or range immediately.

## Original Example Sequence

**Daily joint map**

```text
1. Neck CARs — 2 slow circles each way
2. Shoulder CARs — 2/side
3. Hip CARs — 2/side
4. Ankle CARs — 3/side

Rule:
Move slowly enough to notice restriction. If pinching appears, reduce range and log location.
```

## What Not To Copy

- Do not copy course materials.
- Do not treat CARs as a workout.
- Do not force painful ranges to "open them up."
